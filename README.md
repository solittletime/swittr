# Swittr

Demo app using Node.js, Express.js plus Angular with a service worker.

### Build

Dependencies:

* [Node.js](https://nodejs.org/en/) v8.9.4

Then check out the project and:

```
cd client
npm run build
```

### Running

```
cd server
npm run build
npm start
```

### Using the app

(http://localhost:4300)
