# Swittr

Demo app using Node.js, Express.js plus Angular with a service worker.

### Build

Dependencies:

* [Node.js](https://nodejs.org/en/) v0.12.7 or above

Then check out the project and:

```cd client
ng build --prod
```

### Running

```cd server
node index.js
```

### Using the app

(http://localhost:4300)
